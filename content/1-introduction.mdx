# Prerequisite

* please check the installation on your mac or PC by running <https://docs.flutter.dev/get-started/install>

  ```
  \> flutter doctor
  ```


* Please fork this repo https://github.com/sumithpdd/pomodoro_app and run the command

  ```
  cd .\Pomodoro\
  flutter pub get
  flutter run
  ```

  * use an emulator or a device

    ```
    // IOS
    xcrun simctl list
    open -a Simulator --args -CurrentDeviceUDID <your device UDID>

    // Android
    which adb
    /Users/<user_name>/Library/Android/sdk/tools/emulator -list-avds
    /Users/<user_name>/Library/Android/sdk/tools/emulator -avd <device name>
    ```
